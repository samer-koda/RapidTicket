# RapidTicket Brand Kit v1.0

## Colors
### Primary
- Orange: `#FF5A1F`
- Background darkest: `#121212`
- Background dark: `#1E1E1E`
- Background card: `#2A2A2A`

### Secondary
- Muted Text: gray
- Divider: lighter gray
- Success: green
- Error: red

## Typography
- Font family: **Inter**
- H1: 32px SemiBold
- H2: 22px SemiBold
- Body: 16px Regular
- Micro: 12px Regular

## UI Components
- **Primary button**: orange fill (`#FF5A1F`), white text
- **Secondary button**: outlined (no fill)
- **Chips/modifiers**: pill shaped, `+ Bacon (+$1.99)` / `− Lettuce` style
- **Numpad**: grid layout 1-9 + 0 + backspace
- **Toggle/Tabs**: orange active state (e.g. Tables | Kitchen)
- **Card**: dark bg, item name + orange price
- **Action bar**: large orange CTA button (e.g. "Send 1 Item")

## Icons
- Style: minimal, 2px stroke, rounded joins

## Taglines
- "Speed at Every Table."
- "Orders Without Friction."
- "Fast Service. Zero Noise."

## Brand Voice
- Direct: clear language, no fluff
- Fast: built for speed and flow
- Reliable: confident and operationally strong

## Logo
- Min app icon: 32px (mark only)
- Min header/digital: 120px (horizontal lockup)
- Clear space = height of the "R" stem
